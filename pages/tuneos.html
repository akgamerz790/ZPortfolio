TuneOS

Windows desktop utility for managing and tuning local resources.

Overview

Built with WPF/WinUI and MVVM. Focused on fast navigation, clear settings, and safe defaults.


Tech

C#WPFMVVM

Links

GitHub

Highlights

  • Modular pages with a shared shell and consistent theming.
  • Async operations with cancellation and status feedback.
  • Settings persistence and import/export.

How to run

  • Clone and open in Visual Studio.
  • Build: Release|Any CPU, run F5.
  • Requires .NET Desktop Runtime.
